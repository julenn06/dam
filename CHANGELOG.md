# Registro de Cambios

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-28

### Añadido
- Estructura inicial del repositorio DAM
- Proyectos de primer año (DAM1)
  - Base de datos
  - Entornos de desarrollo
  - Lenguaje de marcas
  - Programación
- Proyectos de segundo año (DAM2)
  - Acceso a datos
  - Desarrollo de interfaces
  - LINQ
  - Multimedia y móviles
  - Procesos y servicios

### Documentación
- README.md completo con descripción del proyecto
- LICENSE.md con licencia MIT
- CONTRIBUTING.md con guías de contribución
- .gitignore para exclusión de archivos

### Características
- Organización por cursos y asignaturas
- Documentación detallada de cada módulo
- Ejemplos y ejercicios prácticos
- Proyectos destacados

## Tipos de Cambios
- `Añadido` para nuevas características
- `Cambiado` para cambios en funcionalidades existentes
- `Obsoleto` para características que serán eliminadas
- `Eliminado` para características eliminadas
- `Arreglado` para corrección de errores
- `Seguridad` para correcciones de vulnerabilidades

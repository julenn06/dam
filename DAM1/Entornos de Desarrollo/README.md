- 📈 **Metodologías de Desarrollo**: Scrum, Agile, Kanban, ciclo de vida del software.

- 🤝 Adquirir competencias en entornos de desarrollo colaborativos.

- 🛠️ Dominar herramientas esenciales para el desarrollo moderno.

Este módulo prepara para trabajar en equipos y mantener código de calidad.

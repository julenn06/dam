# Programación - DAM1

Esta carpeta contiene el módulo de **Programación** del primer curso de DAM, enfocado en el aprendizaje de Java y conceptos de programación orientada a objetos.

## Contenido Principal

### 1.Eval/
- **Ariketa_1.Gaia/** a **Ariketa_5.Gaia/**: Ejercicios por temas.
- **DebugAriketak/**: Ejercicios de depuración.
- **ERRONKA_1/**: Proyecto desafío 1.
- **Julen_Azterketa/**: Examen personal.
- **MaratoiErdia/**: Mitad de maratón (posible proyecto).
- **OsakidetzaAriketa/**: Ejercicio de hospital.
- **Repaso_Azterketa/**: Repaso de exámenes.
- **StringAriketak/**: Ejercicios con strings.
- **StringFuntzioak/**: Funciones de strings.
- **WindowBuilder/**: Desarrollo de GUIs con WindowBuilder.

### 2.Eval/
- **Array_Ariketak/**: Ejercicios con arrays.
- **Array_Bidimentsionalak/**: Arrays bidimensionales.
- **azterketaErrepasoa/** y **azterketaErrepasoa2/**: Repaso de exámenes.
- **AzterketaProg2Julen/** y **Zuzenduta/**: Examen corregido.
- **BD_Konexioa/**: Conexión a bases de datos.
- **Erronka2/**: Desafío 2.
- **Objetuak/**: Programación orientada a objetos.
- **WindowBuilder/**: Más GUIs.

### 3.Eval/
- **ariketaEnuntziatua/**: Ejercicio enunciado.
- **ariketaErrepasoa/**: Repaso.
- **ariketaExcepciones/**: Manejo de excepciones.
- **ariketaMultimedia/**: Aplicaciones multimedia.
- **ariketaTroya/**: Proyecto Troya.
- **aterketa3Eval/**: Examen evaluación 3.
- **azterketaTxuleta/**: Chuleta de examen.
- **mascotas/**: Proyecto de mascotas.
- **Modelo3Ebal/**: Modelo evaluación 3.
- **PosibleExamen/**: Examen posible.
- **RepasoExamen/**: Repaso.

## Tecnologías
- **Java**: Fundamentos, OOP, colecciones, excepciones, I/O.
- **Estructuras de Datos**: Listas, árboles, pilas, colas, algoritmos de ordenación.
- **GUIs**: WindowBuilder, Swing.
- **Acceso a Datos**: JDBC, archivos.
- **Patrones**: Singleton, Factory, MVC.

## Objetivos
- Dominar la programación en Java.
- Aplicar conceptos de OOP y estructuras de datos.
- Desarrollar interfaces gráficas.
- Integrar con bases de datos y archivos.

Este módulo es la base de la programación en DAM.

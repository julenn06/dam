# LINQ & .NET - DAM2

Esta carpeta contiene el módulo de **LINQ & .NET** del segundo curso de DAM, enfocándose en C#, LINQ y Entity Framework.

## Contenido Principal

### 1. Eval/
- **ariketa00/** a **ariketa10/**: Ejercicios de LINQ y .NET.
- **Azterketa1EvalLINQ_Julen/**: Examen evaluación 1.
- **pruebaswagger/**: Pruebas con Swagger.
- **RepasoExamenLINQ/**: Repaso de exámenes.

## Tecnologías
- **C# Avanzado**: Generics, delegates, async/await, reflection.
- **LINQ**: LINQ to Objects, SQL, XML, custom providers.
- **Entity Framework Core**: Code-first, migrations, relationships.
- **ASP.NET Core**: MVC, Web API, middleware, DI, security.
- **API Documentation**: Swagger/OpenAPI.

## Objetivos
- Dominar LINQ para consultas de datos.
- Desarrollar aplicaciones con .NET Core.
- Implementar APIs RESTful.
- Gestionar bases de datos con Entity Framework.

Este módulo introduce el ecosistema .NET.

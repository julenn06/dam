# Multimedia y Móviles - DAM2

Esta carpeta abarca el módulo de **Multimedia y Móviles** del segundo curso de DAM, centrado en desarrollo Android y multimedia.

## Contenido Principal

### 1. Eval/
- **1EbalAzterketaApp/**: App de examen evaluación 1.
- **aplicacionprueba00/**: Aplicación de prueba.
- **ariketa00/** a **ariketa02/**: Ejercicios.
- **Ariketa_5/**: Ejercicio 5.
- **Caendario/**: App de calendario.
- **erronka1mobila/**: Proyecto erronka móvil 1.
- **prueba/**: Pruebas.
- **RepasoPM/**: Repaso de Procesos y Multimedia.
- **SettingApp/**: App de configuraciones.

## Tecnologías
- **Android**: Activities, Fragments, UI con XML/Jetpack Compose, Material Design.
- **Gestión de Datos**: Room, SharedPreferences, APIs.
- **Multimedia**: Camera, media playback, image processing.
- **UX/UI Móvil**: Layouts responsive, gestures, custom views.
- **Características**: Background tasks, notifications, GPS, sensores, Firebase.

## Objetivos
- Desarrollar apps Android nativas.
- Gestionar multimedia y datos en móviles.
- Implementar UI/UX accesibles.
- Publicar apps en stores.

Este módulo se enfoca en aplicaciones móviles.
